# Wendl

Wendl ("AppleHealth to GitHub") reads selected Apple Health data on-device and lets you export it to a GitHub repository you configure.

## Features
- Read selected HealthKit data locally.
- Export data to a user-configured GitHub repository.
- No ads, no analytics, no tracking.

## Requirements
- Xcode (recent version)
- iOS device or simulator with HealthKit support

## Getting Started
1. Open `Wendl.xcodeproj` in Xcode.
2. Select a run destination (device or simulator).
3. Build and run.

## Usage
1. Grant Health permissions when prompted.
2. Configure your GitHub owner/repo and token in Settings.
3. Choose data types and export when ready.
